Short info

Game based on top view shooter type game. The aim is to shoot as many ballons as possible. There is no time limit. The points are granted one for each accurate shoot. The map grid is x= 0-400pt y= 0-300pt. To drop a bomb type command x:y and hit the button Drop. The bomb will smash the ballons at possition x:y and grant you the score.

annotation: center point of a object bomb is not defined, so the top left corner is treated as default

Features list

  • Random generated unique maps and balloons colors are random
  • Explosion sprite type animation
  • Interactive gui
  • Real time scoring system with JSON
  • Scalable game code
  • Mobile friendly game with canvas objects
  • Bespoke design
  • Most recent game development techniques with Ticker for sprite animations and elements rendering
  • Game under 100 lines of code!
